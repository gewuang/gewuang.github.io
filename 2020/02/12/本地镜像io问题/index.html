原因

无方舟本地磁盘为qcow2格式，convert执行触发了读操作

  • 新版方法 convert：[09:39:12~09:44:45] => 5.5分钟
  • 旧版方法 convert+purge：[10:01:40~10:12:09] => 10.5分钟
    • convert触发了读写：[10:01:40~10:04:32] => 3分钟
